Title:
	Modify Profile
Goal: 
	User wishes to modify details of their profile.

Main Success Scenario: 
1)	System displays a number of fields to modify profile data.
2)	User chooses a field.
3) 	User changes the data.
4)	User decides to save changes
4)	System modifies the User's profile. 
5)	System returns to profile page.

Extensions:
3a)	User decides to cancel changes. 
	.1:	System returns to profile page.
